## 🎯 Objective

Model and forecast financial time series while accounting for non-stationarity and volatility clustering.

---

## 🧠 Approach

- Conducted ADF test for stationarity  
- Applied differencing where required  
- Built ARIMA models for forecasting  
- Modeled volatility using GARCH  

---

## 📊 Results

- Achieved stationarity via transformation  
- ARIMA produced consistent forecasts  
- GARCH captured time-varying volatility
