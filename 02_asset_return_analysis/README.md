## 🎯 Objective

Analyze statistical properties of financial returns to assess risk and distribution behavior.

---

## 🧠 Approach

- Computed descriptive statistics  
- Evaluated distribution shape (skewness, kurtosis)  
- Analyzed rolling volatility  
- Measured drawdowns  

---

## 📊 Findings

- Returns exhibit **fat tails and skewness**  
- Volatility clustering observed  
- Significant downside risk detected
